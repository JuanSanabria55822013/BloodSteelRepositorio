README — Bloodsteel (Build Jugable)

Versión: 1.0
Motor: Unreal Engine 5.6
Plataforma: Windows 64-bit

INSTALACIÓN

Descarga el archivo .zip proporcionado.

Descomprime el contenido en cualquier carpeta.

Abre la carpeta descomprimida.

Ejecuta el archivo: Bloodsteel.exe

No se requieren plugins adicionales ni instalaciones externas.

CONTROLES BÁSICOS
WASD: Movimiento
Espacio: Salto / Doble salto
Shift: Dash
Mouse 1: Disparo
R: Recargar
E: Interactuar
P / Esc: Menú
1: Pistola Electrica
2: Metralleta
3: Escopeta
4: Lanzacabezas

REQUISITOS MÍNIMOS
Sistema operativo: Windows 10 o 11
Procesador: Intel i5 o equivalente
GPU: GTX 1060 / RX 580
RAM: 8 GB
Almacenamiento: 4–8 GB libres

NOTAS
Esta es una versión de prueba y puede presentar errores.
Para un mejor rendimiento, se recomienda ejecutarlo en pantalla completa.
